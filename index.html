HTML & CSS Cheat Sheet

What is HTML?

HTML stands for Hypertext Markup Language. It is the first step to build a website and its the skeleton of all web pages. HTML provides structure to the content on a website, including, text, images, and videos.

Markup language is a computer language that defines the structure and presentation of raw text. Hypertext is a text displayed on a computer or device that provides access to other text through links, also known as Hyperlink

What is CSS?

CSS stands for Cascading Stylesheet. It is the language that provide styles to the content of HTML page. This style includes colors, fonts, positioning, layout, and many more!

Why do we need to separate language for content and presentation?

This is because it is easier to maintain when each section has a clearly differentiated problems that it is trying to solve.

HTML Element
Tag Element Name Description
<!-- . . . --> Comment To insert comments in the source code. It does not displayed in the browsers. Makes it easier to read code if there is a comment to explain it.
<!DOCTYPE HTML> Opening HTML documents must start with a <!DOCTYPE> declaration. It is an information to the browser about what document type to expect.
<html>. . .</html> HTML Tag Represent the root of an HTML document. Without these tags, it's impossible that browsers could incorrectly interpret your HTML code.
<head>. . .</head> Head A container for metadata (data about the HTML document) and is placed between <html> tag and <body> tag.
The following elements can go inside <head> element: <title>, <style>, <base>, <link>, <meta>, <script>, and <non-script>.
<title>. . .</title> Title The title of the document. Must be text-only, and shown in the browser's title bar or in the page's tab. It is required in HTML documents!, because it's very important for Search Engine Optimization (SEO) algorithms to decide the order when listing pages in search results.
<link rel="stylesheet" type="text/css" href="styles.css"> Link The relationship between the current document and an external source. It is most often used to link an external style sheets to your website.
It is an empty element, it contains attributes only.
<a href="link">. . .</a> Link It is a hyperlink, which is used to link from one page to another.
<body>. . .</body> Body Document's Body. It contains all of HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, and many more!
<footer>. . .</footer> Footer The content at the bottom of the subject information.
It contains information such as; Contact Information, Copyright Information, Term of Use, Site Map, Reference to Top of Page Links, etc.
<article>. . .</article> Article Specifies independent, self-contained content. It holds content that makes sense on its own. It will help someone with screen reader to understand where the aricle content begins and ends.
<aside>. . .</aside> Aside It used to mark additional information that can enhance another element, but it not required in order to understand the main content.
Some common uses: Bibliographies, endnotes, comments, pull quotes, editorial sidebars, additional information, etc.
<audio controls> <source src="link.mp3" type="audio/mp3"> </audio> Audio It is used to embed sound content in a document, such as music or other audio streams.
<button>. . .</button> Button Clickable button.
<caption>. . .</caption> Caption Table of caption.
Must be inserted after the <table> tag.
<div>. . .</div> Division A division or a section in an HTML document. It is used as a container for HTML elements - which is then styled with CSS.
<em>. . .</em> Emphasized Text It is to emphasized text and will typically displayed in italic.
<embed type="animation.gif" src="link.gif">. . .</embed> Embed To incorporate media content into a page, which then can embed any media content including videos, audio files, and gifs from an external source.
<figure>. . .</figure> Figure Is an element used to encapsulate media such as images, illustrations, diagrams, code snipper, etc.
<figcation>. . .</figcaption> Figure Caption Is an element used to describe (or add caption) the media in the <figure> tag. Can be placed as the first or last child of the <figure> tag.
<h1>. . .</h1> to <h6>. . .</h6> Headings HTML headings which is similar to heading in other types of media.
There are six different headings tag (h1 to h6). <h1> defines the most important heading, whereas <h6> defines the least important heading.
<header>. . .</header> Header It is a container for either navigational links or introductory content containing <h1> to <h6> headings, logo or icon, and authorship information.
<img src="img.jpg" alt="accessible text for the image"> Image Allows you to adds an image to a web page. Images are not technically inserted into a web page; images are linked to a web pages. <img> is a self-closing tag.
<main>. . .</main> Main Encapsulate the dominant content within a web page. The content inside should be unique to the document. It should not contain any content that is repeated across documents.
The <main> element contains: <article>, <aside>, <footer>, <header>, or <nav>
<nav>. . .</nav> Navigational Links A block of navigation links such as menus and tables of content. It is intended only for major blocks of navigational links.
<ol>. . .</ol> Ordered List Displayed content in an easy-to-read list. It can be numerical or alphabetical.
<ul>. . .</ul> Unordered List Similar to Ordered List, except that each list item have no particular order. Unordered List outlines individual list item with a Bullet Point.
<li>. . .</li> List Item Used inside Order List (<ol>), Unorder List (<ul>), and in menu lists (<menu>).
<p>. . .</p> Paragraph Paragraph of text. It is a block-level elements and will automatically close if another block-level element is parsed before the closing tag
<section>. . .</section> Section Elemets in a document, such as: chapters, headings, or any other area of document with the same theme.
<span>. . .</span> Span A section in a document. It is best to use this element when you want to target a specific piece of content.
<strong>. . .</strong> Strong Making text with strong importance. The content inside is typically displayed in bold.
<video src="video.mp4" controls>. . .</video> Video To embed video content in a document, such as a movie clip or other video stream, makes it clear that a developer is attempting to display a video to the user.
HTML Tables
<table>. . .</table> Table HTML Table. It consist of one <table> element and one or more <tr>, <th>, and <td> element.
An HTML table may also include <caption>, <colgroup>, <thead>, <tbody>, and <tfoot>
<thead>. . .</thead> Table Head Tag used to group header content in an HTML table. This will make it easier to read and manage if the table is sectioned off.
<tbody>. . .</tbody> Table Body Tag used to group body content in an HTML table. This will make it easier to read and manage if the table is sectioned off. The elemet should contain all of the tables data.
<tfoot>. . .</tfoot> Table Footer Tag used to group footer content in an HTML table. The bottom part of a long table. Footer often used to contain sums, differences, and other data result.
<tr>. . .</tr> Table Rows A row in an HTML table.
<th>. . .</th> Table Headings A header cell in an HTML table, it contains header information. The text in <th> elements are bold and centered by default.
<td>. . .</td> Table Data A standard data cell in an HTML table, it contains data. The text are regular and left-aligned by default.
<th colspan="number">. . .</th> Columns Spanning Number of columns a header cell should span. The value attributes ("number") must be a positive number, it specifies the number of columns that the cell fills.
This element attributed to <th> and <td> element.
<th rowspan="number">. . .</th> Rows Spanning Number of rows a header cell should span. The value attributes ("number") must be a positive number, it specifies the number of rows that the cell fills.
This element attributed to <th> and <td> element.
<th scope="col">. . .</th> Scope It specifies whether a header cell is a header for a column, row, or group of columns or rows. It has no visual effect in ordinary web browsers, but can be used by screen readers.
